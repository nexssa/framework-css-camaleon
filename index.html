Sistema de columnas

El sistema adaptable se basa en un esquema de filas de 12 columnas. para los contenedores se utilizan las clases kontainer y kontainer-full para utilizar parte de la pantalla o todo el ancho según el tipo de dispositivo.

Las resoluciones o puntos de ruptura que se pueden utilizar son:

  • kol-s-x
  • kol-m-x
  • kol-x
  • kol-l-x

Se utiliza s para dispositivos de baja resolución, m para dispositivos de mediana resolución (smartphones y tablets), l para dispositivos con mayor resolución (large) y sin identificador de forma predeterminada para equipos de escritorio estándares (ej. kol-12).
